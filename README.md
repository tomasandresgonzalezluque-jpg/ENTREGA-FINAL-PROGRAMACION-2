A qué apuntar: ¿Felicidad o tristeza?
Proyecto Programación II – 2025-2

Este repositorio contiene el proyecto final del curso Programación II, realizado por los estudiantes Tomás González, Sebastián Ardila, Juan Otálora y Fernando Rodríguez.

El trabajo desarrolla un análisis aplicado en Python, utilizando Google Colab para procesar datos y explorar la pregunta central del proyecto: la relación entre felicidad, tristeza y sus patrones asociados.

Cómo usar este repositorio

Abrir el archivo .ipynb directamente en GitHub o descargarlo.

Para ejecutar el proyecto, abrir el notebook en Google Colab.

Instalar las librerías necesarias según se indican en el código.

Ejecutar las celdas en orden para reproducir el análisis completo.

Objetivo del proyecto

Aplicar técnicas de programación y análisis de datos para abordar la pregunta planteada: ¿A qué apuntar, la felicidad o la tristeza?
El proyecto examina patrones y relaciones presentes en los datos con el fin de obtener conclusiones fundamentadas en evidencia.
